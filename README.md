# The carbonate clumped isotope calulcation by Python

## Author: Keran Li

### Introduction

This repository contains the code files for the carbonate clumped isotope calculation by Python. 

### Dependencies

* Python 3.10
* Jupyter Notebook
* Numpy
* Scipy
* Matplotlib
* Pandas
